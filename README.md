# BlogGenie AI
Automatically generates and publishes SEO optimised blog posts using AI.
BlogGenie AI is a WordPress plugin that generates blog articles using AI and publishes them automatically.

---

## Features
- AI powered blog generation
- SEO optimised content
- Automatic featured image generation
- Scheduled publishing
- Lightweight WordPress plugin

---

## Installation
1. Download the plugin
2. Upload to /wp-content/plugins/
3. Activate BlogGenie AI
4. Configure API keys

---

## Requirements
- WordPress 6+
- PHP 7.4+
- OpenAI API

---

## Usage
1. Enter a topic or keyword
2. BlogGenie AI generates the article
3. Image is generated automatically
4. Post is scheduled or published
